# Лабораторна робота №7: Програмна реалізація оброблення масивів даних та символьної інформації за стандартом Unicode

Цей репозиторій містить результати виконання лабораторної роботи №7 варіанту 9 з дисципліни "Базові методи та технології програмування" (БМТП). Проєкт реалізований мовою C із забезпеченням кросплатформної сумісності (Windows/Linux) та підтримкою кодування UTF-8.

## Виконавець:
* **Студент:** Бурейко Олександр Григорович
* **Група:** КБ-25
* **Спеціальність:** Кібербезпека та захист інформації
* **Університет:** ЦНТУ

## Структура проєкту:
* **prj/** — вихідний код програм:
  * **task_7_1/** — обробка символьної інформації (пошук складів "ма", вивід приголосних).
  * **task_7_2/** — обробка числових масивів (пошук екстремумів, обчислення суми та різниці).
* **Software/** — скомпільовані виконувані файли (.exe для Windows).
* **TestSuite/** — артефакти системного тестування (20 тест-кейсів для кожної задачі).

## Технологічний стек:
* **Мова:** C (C18).
* **Середовище:** Code::Blocks IDE.
* **Версійність:** Git+GitHub.
* **Локалізація:** Unicode (UTF-8) з підтримкою кириличного вводу-виводу.
